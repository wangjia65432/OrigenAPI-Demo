18  * \brief Create a view of a TransitionCoeff type
19  *
20  * GridView types allow for convenient translation of ORIGEN data types
21  * intended for interpolation (here, TransitionCoeff - the burnup-dependent
22  * cross-sections on each ORIGEN Library) onto GridData responses. This method
23  * creates a single TransitionCoeff GridData "View"; a series of "views"
24  * are used to populate response extents - i.e., each TransitionCoeff
25  * corresponds to a different interpolable grid extent, such as burnup,
