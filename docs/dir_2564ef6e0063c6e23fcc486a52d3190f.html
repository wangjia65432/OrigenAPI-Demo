file  LibraryIO_bof.cpp
 This file deals with the intput and output of Origen library contents to and from the file on disk. The data is put into BOF data types and tagged with three-letter names for the burnup invariant data and two-letter names for the burnup dependent data. The read function pulls from the file and rebuilds the library object. The write function puts every part of the library into the appropriate formats and adds to the object written to the file.
