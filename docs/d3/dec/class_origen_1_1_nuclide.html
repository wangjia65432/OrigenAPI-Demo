Detailed Description

Defines the nuclide interface for the TransitionSystem.

The nuclide, in the context of the transition system, manages five main pieces of data:

  1. identification
  2. decay constant
  3. total loss xs
  4. loss channels (Channel)
  5. gain transitions (Transition)

The nuclide also has

  1. a strong pointer to a Species
  2. a weak pointer to a TransitionSystem
  3. a number of this nuclide (to calculate reaction rates)
See also
Channel, Transition, Species, TransitionSystem
